# 快速修复域名白名单错误

## 错误信息
```
errMsg: "request:fail url not in domain list"
```

## 立即解决（3步）

### 步骤 1：打开微信开发者工具设置

在微信开发者工具中：
1. 点击右上角的 **`详情`** 按钮（或按 `Cmd/Ctrl + ,`）

### 步骤 2：勾选不校验域名

在打开的设置窗口中：
1. 点击 **`本地设置`** 标签页
2. 找到 **`不校验合法域名、web-view（业务域名）、TLS 版本以及 HTTPS 证书`**
3. ✅ **勾选这个选项**

### 步骤 3：重新编译

1. 关闭设置窗口
2. 点击 **`编译`** 按钮重新编译
3. 测试网络请求

## 设置位置图示

```
微信开发者工具
  └─ 右上角 "详情" 按钮
      └─ "本地设置" 标签页
          └─ ☑️ 不校验合法域名、web-view（业务域名）、TLS 版本以及 HTTPS 证书
```

## 验证

配置后，网络请求应该可以正常发送，不再出现 "url not in domain list" 错误。

## 注意事项

- ⚠️ 这个设置**仅用于开发环境**
- ⚠️ 正式发布前需要配置真实的服务器域名
- ⚠️ 生产环境必须使用 HTTPS 域名

## 如果还是不行

1. 检查后端服务是否启动（默认端口 8080）
2. 检查 `common/utils/request.js` 中的 `baseUrl` 配置
3. 查看详细解决方案：`解决域名白名单错误.md`
